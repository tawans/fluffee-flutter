📄 요구사항 정의서: 프랜차이즈 카페 모바일 앱
1. 개요
목적: 실제 프랜차이즈 카페에서 운영할 수 있는 수준의 모바일 앱을 구현하여 포트폴리오로 활용한다.

벤치마크: 바나프레소 앱

대상 사용자: 프랜차이즈 카페 고객

2. 주요 기능
2.1. 홈
브랜드 소개: 카페의 스토리, 철학, 사진 등 소개

이벤트/프로모션: 진행중인 행사 및 혜택 배너

추천 메뉴/인기 메뉴: 시즌별, 판매량 기반으로 노출

2.2. 메뉴
카테고리별 메뉴 리스트

예) 커피, 티, 스무디, 디저트 등

메뉴 상세 보기

이름, 가격, 설명, 사진, 칼로리 등 상세 정보

2.3. 매장 검색
현재 위치 기반으로 가까운 매장 표시 (리스트 & 지도)

매장 상세 정보

주소, 전화번호, 영업시간, 혼잡도

자주 가는 매장 등록/해제

2.4. 주문
메뉴 선택 및 장바구니 담기

픽업할 매장 선택

주문하기

주문 접수 → 제작 중 → 픽업 가능 상태 표시

주문 후 5~10분 랜덤 시간 경과 후 자동으로 상태 변경

주문 취소는 제작 시작 전까지만 가능 (선택)

2.5. 마이 페이지
내 프로필 정보

이름, 연락처, (선택) 프로필 사진

최근 주문 내역

메뉴, 매장, 주문 시간, 상태

자주 가는 매장 목록

(선택) 쿠폰/적립금 정보

3. 주문 상태 흐름
✅ 주문 접수 → 🔄 제작 중 → 📦 픽업 가능

주문 완료 후 5~10분 사이에 상태가 제작 중으로 전환되고, 이후 픽업 가능으로 바뀜

상태는 사용자가 새로고침하거나 알림으로 확인 가능

4. 비기능적 요구사항
실제 매장에서 사용할 수 있는 듯한 자연스러운 플로우

직관적이고 심플한 UX/UI

모바일 환경 최적화

모든 주요 기능은 로그인 없이 둘러볼 수 있고, 주문/내역 관리에는 로그인 필요

5. 화면 구성
화면	주요 내용
홈	카페 소개, 이벤트, 추천 메뉴
메뉴	카테고리별 메뉴, 상세 정보
매장	지도 & 리스트, 매장 상세, 자주 가는 매장 등록
주문	장바구니, 매장 선택, 결제, 상태 확인
마이 페이지	내 정보, 주문 내역, 자주 가는 매장

6. 확장 가능 기능 (선택사항)
푸시 알림: 주문 상태 변경 시 알림 발송

리뷰 작성: 메뉴/매장 리뷰 기능

관리자 모드: 메뉴, 매장, 이벤트 관리
